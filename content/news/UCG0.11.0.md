This is a relatively mall update as we've just returned from a break. Expect weekly updates to continue. Apologies for the wait, hopefully you had a fun holidays and new years!

# Patch Notes

## Added / Removed

- Added battle log messages for unit deaths.

- Added a score display to achievements in the achievements panel. Achievements are now ordered depending on score.

- Added icons to main menu buttons.

- Added a warning when trying to start a battle with unequipped relics.

## Edits

- Updated the effect description of the relic "Desperation" to better match it's actual effect.

Updated description: "At the start of the turn, if the owner is below 25% of their maximum HP, reduce the cost of all cards in their hand by 1. Cost cannot be reduced below 0 by this effect."

Previous description: "At the start of the turn, if the owner is below 25% of their maximum HP, their cards cost 1 less MP to play. Cost cannot be reduced below 0 by this effect."

- Updated the copyright year.

- Updated the color of shield health to white.

- Updated the modifier run shop modifier panel slightly.

## Fixes

- Fixed an issue where the starting deck had three copies of the card 'Trick' by accident.

- Fixed some small grammar issues in relic descriptions.
